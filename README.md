## 简介

在一些团队技术管理工作中，比如培训，设计评审，代码复查等，使用技术博客来管理是非常方便的，网上找了一波，发现很多要么样式或功能达不到我想要的，要么就不是全部开源，所以就萌生了自己写一个的想法，本项目基于MIT协议，所有代码完全开源。
